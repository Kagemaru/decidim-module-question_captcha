# Decidim::QuestionCaptcha

Question based captcha for Decidim.

## Usage

QuestionCaptcha will be available as a Component for a Participatory
Space.

## Installation

Add this line to your application's Gemfile:

```ruby
gem "decidim-question_captcha"
```

And then execute:

```bash
bundle
```
